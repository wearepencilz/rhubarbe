# Preorder Operations — CMS User Guide

## Availability Patterns

Navigate to **Admin → Availability Patterns** to manage ordering rules.

Pattern types:
- **Recurring Weekly**: Orders open/close on specific days each week (e.g., Mon–Thu for Saturday pickup)
- **Rolling Lead-Time**: Orders accepted with a minimum lead time (e.g., 48h before pickup)
- **One-Off Scheduled**: Fixed date range for special events

Each pattern defines: order timing, pickup rules, slot configuration, quantity rules, and bilingual customer messaging.

## Pickup Locations

Navigate to **Admin → Pickup Locations** to manage physical pickup points.

Each location has: internal name, bilingual public label, address, contact details, map link, and bilingual pickup instructions. Drag to reorder.

## Slot Templates

Navigate to **Admin → Slot Templates** to define time slot configurations.

Each template defines: start time, end time, interval (minutes), and default per-slot capacity.

## Menu Weeks

Navigate to **Admin → Menu Weeks** to manage weekly menus.

Each menu week has: internal title, bilingual week label, launch date, order cutoff, pickup date range, featured products, and bilingual intro copy.

## Product Availability Tab

In the **Product Editor**, the **Availability** tab controls:
- **Selling Mode**: online orderable, pickup only, availability mode
- **Order Rules**: min/max quantity, quantity step, inventory mode
- **Pickup Rules**: pickup required, location restriction, date/slot selection
- **Availability Windows**: date-specific overrides for the product

## Order Operations

Navigate to **Admin → Orders** to manage customer orders.

Features:
- Filter by pickup date, location, status
- Search by order number or customer name
- View order details with all pickup information
- Mark orders as fulfilled or cancelled
- **Generate Prep Sheet**: Aggregated product quantities by date range
- **Generate Pickup List**: Customer orders grouped by time slot for a specific date/location
- **Manage Slot Capacity**: Monitor and adjust slot capacity in real-time

## Reports

Both prep sheets and pickup lists can be exported as CSV for printing or further processing.
